# biometric_iot_bridge — VS Code Extension

Flutter biometric auth & secure MQTT IoT bridge — snippets, docs, pub.dev integration.

## Features
- 📖 One-click docs, pub.dev, GitHub access
- ⚡ 7 Dart snippets (bib-flow, bib-widget, bib-verify...)
- 🖥️ Sidebar panel with package info
- 🔘 Status bar quick-pick menu
- 📦 Auto-detects biometric_iot_bridge in pubspec.yaml

## Snippets
| Prefix | Description |
|---|---|
| `bib-import` | Import statement |
| `bib-init` | Initialize bridge |
| `bib-verify` | Verify biometrics |
| `bib-token` | Generate secure token |
| `bib-signal` | Send MQTT signal |
| `bib-flow` | Complete auth flow |
| `bib-widget` | Full StatefulWidget |
